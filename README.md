
# Vijaywada Ayurvedic Hospital Portfolio

A React + Vite portfolio/business website concept for Vijaywada Ayurvedic Hospital.

## Tech Stack
- React.js
- Vite
- Framer Motion
- Lucide Icons
- CSS3 responsive design

## Run Locally
```bash
npm install
npm run dev
```

## Sections
- Hero section
- Hospital services
- Patient journey
- Wireframe project screens
- AI-assisted UX workflow
- Design system
- Contact/footer
